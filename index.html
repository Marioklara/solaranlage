Was sollte man vor einer PV-Anfrage wissen?

Die passende Größe einer Photovoltaikanlage hängt von Dachfläche, Ausrichtung, Stromverbrauch und zukünftigen Verbrauchern ab.

Ein Batteriespeicher kann den Eigenverbrauch erhöhen. Ob er wirtschaftlich sinnvoll ist, hängt vom individuellen Verbrauch ab.

Auch E-Auto und Wärmepumpe können die Planung beeinflussen.

Hilfreiche Angaben für eine Anfrage

✓ Postleitzahl

✓ Eigentumsverhältnis

✓ ungefährer Jahresstromverbrauch

✓ Dachgröße und Dachform

✓ Interesse an Batteriespeicher

✓ E-Auto oder Wärmepumpe geplant
