# PV-Angebot24

Statische Landingpage und SEO-Seiten für einen Solar-/Photovoltaik-Vergleich.

## Enthalten

- Startseite mit klarem Angebot: "Solaranlage kostenlos vergleichen"
- PV-Rechner mit PLZ, Hausart, Dachgröße, Stromverbrauch, Personen, Strompreis und Speicherwunsch
- Unterseiten für Kosten, Vergleich, Stromspeicher, Wärmepumpe, Förderung, Rechner und Ratgeber
- Lokale SEO-Seiten für Heidelberg, Mannheim, Schwetzingen und Karlsruhe
- `sitemap.xml` und `robots.txt`
- Kontaktformular mit Datenschutz-Zustimmung über FormSubmit
- Keine erfundenen Bewertungen, Fake-Siegel oder Fake-Partnerpreise

## Google Search Console

Nach Veröffentlichung:

1. Property für die Domain oder GitHub-Pages-URL in Google Search Console anlegen.
2. Website bestätigen.
3. `sitemap.xml` einreichen.
4. Wichtige Seiten zur Indexierung beantragen.

## Spätere Schritte

- Echte Partnerprogramme prüfen und erst nach Annahme Tracking-Links einbauen.
- Datenschutzerklärung erweitern, falls Analytics, Cookies oder Affiliate-Tracking genutzt werden.
- Optional ein Backend für Leads, Partnerzuordnung, E-Mail-Benachrichtigungen und Admin-Bereich bauen.
