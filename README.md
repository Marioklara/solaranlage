# PV-Angebot24

Einfache Landingpage für Solar-/Photovoltaik-Vergleich.

## Hochladen auf GitHub
1. Neues Repository erstellen
2. Diese Dateien hochladen: `index.html` und `README.md`
3. Settings → Pages
4. Branch `main` auswählen
5. Speichern

## Wichtig
- Impressum ist mit deinen Angaben eingefügt.
- Kontaktformular nutzt FormSubmit.
- Beim ersten Formular-Test musst du die Bestätigungs-Mail von FormSubmit bestätigen.
- Anbieterlinks sind normale Links, noch keine Affiliate-Links.
